𝜋 Test

About the test

What is it?

"𝜋 Test" is a simple and popular mathematical test which can be written in any programming language (like Java, Javascript, C, VisualBasic, etc.) having randomized function which generating numbers from a specified range.

How does it work?

The main idea is getting information about how big part of total generated points will be inside inscribed circle of the square.

No matter what is the length of the square edge ("a"), result of division inscribed circle area (S1) by square area (S2) is always the same and is equal 𝜋/4.

Because points must be evenly distributed, we can expect that during the test 𝜋/4 will be the permanent value (in theory) and because of that the calculated value multiply by 4 will give us "full" 𝜋 number.

Additional info

This "𝜋 Test" is based on ES6 Javascript (click for ES5 version).

Here we got "step by step" method - every point (pixel) will be displayed on the screen (canvas) instantly after it has been calculated. There is big disadventage here - because we have got always for every point pair of "calculate + display" processes and screen refreshing is 60Hz we need at least 16 seconds for 1000 points (1 sec/60 * 1000).

Because test is much slower I give smaller number of points, but as a bonus there is an extra option in the table (percent of points already tested).

Main part of the "𝜋 Test"

Depending on situation sometimes we need numerical or graphical presentation of the result. Below I put both of them and it is up to you which one (or both) you will take.

In the table and inside the square always appear the final test values. Square is 300x300 points (px) large and every black/red dot is a single result from the "Pi test".

Choose from the "TEST" list number of the points and click "Start" button.
