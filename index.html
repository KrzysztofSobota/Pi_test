Random generated numbers (Pi Test)

"Pi test" is simple and popular test checking how good is programming code engine (in Java, Javascript, C, VisualBasic, etc.) which generate pseudo-randomize numbers.The main idea is get information about how big part from total generated points will be inside inscribed circle of the square.

No matter what is the length of the square side, result of division circle area (S1) by square area (S2) is always the same and is equal 𝜋/4.



Because points must be localized evenly, we can expect that during the test 𝜋/4 will be the permanent value (in theory) and because of that the calculated value multiply by 4 will give us "full" 𝜋 number.

Description of working code:
  1. Progress - procent of total points tested
  2. Points tested - how many points was recently tested
  3. Points inside circle - how many points are inside circle area
  4. Estimated 𝜋 value - 𝜋 calculated during the test (recently value)
Choose from the "TEST" list number of the points and click button "Start"
